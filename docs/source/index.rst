Welcome to Introduction to Causality documentation!
===================================

**do-causality** is a Python library for folks who are interested in using probabilisitic programming language, numpry for causality.

.. note::

   This project is under active development.

Installation
------------

To use do-causality, first install it using pip:

.. code-block:: console

   (.venv) $ pip install do-causality
